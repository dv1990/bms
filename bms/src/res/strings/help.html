How to run this program

Readme before you tear apart your PC

  1. Connect Arduino via USB to the PC
  2. Click on the .jar package or start the .jar via the command java -Xmx128m -XX:+UseSerialGC -jar theNameOfTheJar.jar
  3. The software automatically connects to the first Arduino it detects.
