GUI BMS manager

How to run this program

Readme before you tear apart your PC

  1. Connect Arduino via USB to the PC
  2. Click on the .jar package or start the .jar via the command java -Xmx128m -XX:+UseSerialGC -jar bms.jar
  3. The software automatically connects to the first Arduino it detects, so make sure there is only one Arduino connected to you PC

Bms manager

Tune here the settings of the GUI and the Arduino

Battery pack visualizer

Shows data of single cells of segments

  1. Left mouse click on a cell shows the real time plot of temperature and voltage of single cell
  2. Right mouse click on a cell shows the real time plot of temperature and voltage of all cells in the segment it belongs

Logging window

Shows logs coming from Arduino

  1. Left mouse click on the window sets the window to automatically scroll on new data
  2. Right mouse click on the window sets autoscroll off

Chart window

Shows real-time plots of average voltage and temperature in battery pack
